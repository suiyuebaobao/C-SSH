## 这是什么

Creation-SSH 是一款**跨平台的 SSH 运维产品**（桌面优先 + 移动伴侣），在服务器上常驻一个轻量 agent，把"原生客户端体验、服务端 tmux 持久化会话、常驻监控"三件事缝成一个产品——不是又一个网页面板，也不是又一个普通 SSH 终端。

> 一句话:**Xshell 级的原生客户端 × 常驻 agent 的结构化能力 × tmux 级的持久化,三合一。**

## 核心能力

- 🖥️ **双模式终端** —— 持久化终端(agent 直驱 tmux,断线/换设备/升级都不丢会话)+ 普通直连终端(未装 agent 也能用,关闭即销毁)
- 📊 **常驻监控** —— CPU/内存/磁盘/网络/磁盘 IO/Top 进程实时六卡 + 历史曲线,服务器侧持续采集
- 📁 **文件管理器** —— 浏览/增删改查/内置编辑器/分块传输/断点续传/整目录压缩下载
- 🔌 **端口映射** —— SSH 本地转发,默认绑定本机
- 🧩 **应用中心** —— Docker 容器/镜像一键安装、systemd 服务管理、应用商城一键装 Docker
- 🛠️ **服务器系统管理** —— 进程/防火墙端口/改 SSH 密码,破坏性操作二次确认
- 🤖 **内置 AI 助手** —— 长在服务器里的 AI 运维/编程助手:只读诊断、写文件、跑命令、把长任务丢进持久化窗口,五档权限模式 + 执行确认 gate,可接 OpenAI 兼容与 Anthropic(Claude)等模型
- 🌍 **多语言** —— 简体/繁体中文、英语、法语、俄语、韩语、西班牙语、德语、葡萄牙语
- 🔐 **凭据本地加密** —— 私钥/密码进本地加密保险库,绝不上传服务器
